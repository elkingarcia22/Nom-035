Crear botón UBITS 

Crea un botón UBITS con estas características:
- Texto: [especifica texto]
- Variante: [primary/secondary/tertiary]
- Tamaño: [sm/md/lg]
- Icono: [especifica icono FontAwesome]
- Ubicación: [describe en que parte del layout debe estar el botón]

IMPORTANTE: Usa EXACTAMENTE esta estructura:
<button class="ubits-button ubits-button--[variante] ubits-button--[tamaño]">
    <i class="far fa-[icono]"></i>
    <span>[texto]</span>
</button>

IMPORTA: components/button.css

Crear nuevo botón 

Crea un botón que diga "[texto_del_boton]" con estilo [primary, secondary o tertiary] y tamaño [small, medium o large] en [ubicación donde quieres el botón].

Agregar indicador al botón 

Agrega un punto rojo de notificación al botón [nombre_del_boton].

Cambiar acción del botón 

Haz que el botón [nombre_del_boton] al hacer clic [describe qué acción debe realizar].

Convertir a botón de solo icono 

Convierte el botón [nombre_del_boton] a la variante de solo icono. Quita el texto y usa solo el icono [nombre_del_icono]. Agrega la clase ubits-button--icon-only.

Documentación técnica

Información técnica detallada para implementar el componente Button.

Mostrar

Estructura del componente 

<button class="ubits-button ubits-button--primary ubits-button--md">
  <i class="far fa-check"></i>
  <span>Botón primario</span>
</button>

Clases disponibles 

/* Variantes */
.ubits-button--primary    /* Botón principal (azul) */
.ubits-button--secondary  /* Botón secundario (gris) */
.ubits-button--tertiary   /* Botón terciario (transparente) */

/* Tamaños */
.ubits-button--sm        /* Pequeño (13px, 32px alto) */
.ubits-button--md        /* Mediano (16px, 40px alto) */
.ubits-button--lg        /* Grande (20px, 48px alto) */

/* Modificadores */
.ubits-button--icon-only /* Solo icono, sin texto */
  /* Small: 32x32px, Medium: 40x40px, Large: 48x48px */
  /* Uso: Acciones secundarias, eliminar, controles compactos */

Cómo implementar

🔧 REQUISITOS OBLIGATORIOS:

CSS Requerido:
• components/button.css - Estilos del componente
• fontawesome-icons.css - Iconos FontAwesome
• ubits-colors.css - Tokens de color UBITS
• ubits-typography.css - Tipografía UBITS

JavaScript: NO requerido - Componente puramente CSS

Renderizado: HTML directo con clases UBITS 
<!-- IMPLEMENTACIÓN BÁSICA -->
<!-- Botón primario mediano -->
<button class="ubits-button ubits-button--primary ubits-button--md">
  <i class="far fa-check"></i>
  <span>Confirmar</span>
</button>

<!-- Botón secundario pequeño -->
<button class="ubits-button ubits-button--secondary ubits-button--sm">
  <span>Cancelar</span>
</button>

<!-- Botón terciario grande -->
<button class="ubits-button ubits-button--tertiary ubits-button--lg">
  <i class="far fa-plus"></i>
  <span>Agregar elemento</span>
</button>

<!-- Botón de solo icono -->
<button class="ubits-button ubits-button--tertiary ubits-button--sm ubits-button--icon-only">
  <i class="far fa-trash"></i>
</button>
