Características generales

El componente Input de UBITS es un sistema completo de campos de entrada que cubre todos los casos de uso comunes en aplicaciones web modernas.

11 Tipos de Input
text, email, password, number, tel, url, select, textarea, search, autocomplete, calendar
3 Tamaños
Small (32px), Medium (40px), Large (48px)
5 Estados
default, hover, focus, invalid, disabled

Cómo implementar

🔧 REQUISITOS OBLIGATORIOS:

CSS Requerido:
• components/input.css - Estilos del componente
• fontawesome-icons.css - Iconos FontAwesome
• ubits-colors.css - Tokens de color UBITS
• ubits-typography.css - Tipografía UBITS

JavaScript Requerido:
• components/input.js - Función createInput()

Renderizado: createInput() function con containerId 
<!-- 1. IMPORTAR ARCHIVOS REQUERIDOS -->
<link rel="stylesheet" href="ubits-colors.css">
<link rel="stylesheet" href="ubits-typography.css">
<link rel="stylesheet" href="fontawesome-icons.css">
<link rel="stylesheet" href="components/input.css">
<script src="components/input.js"></script>

<!-- 2. CREAR CONTENEDOR -->
<div id="mi-input-container"></div>

<!-- 3. IMPLEMENTAR INPUT BÁSICO -->
<script>
document.addEventListener('DOMContentLoaded', function() {
    createInput({
        containerId: 'mi-input-container',
        label: 'Email',
        placeholder: 'correo@ejemplo.com',
        type: 'email',
        size: 'md',
        leftIcon: 'fa-envelope',
        helperText: 'Ingresa tu email válido',
        showHelper: true,
        mandatory: true,
        mandatoryType: 'obligatorio'
    });
});
</script>

<!-- 4. VALIDACIÓN MANUAL (OBLIGATORIA) -->
<script>
setTimeout(() => {
    const input = document.querySelector('#mi-input-container input');
    if (input) {
        input.addEventListener('input', function() {
            const value = this.value;
            if (value.includes('@') && value.includes('.')) {
                this.style.borderColor = 'var(--ubits-border-1)';
                this.style.borderWidth = '1px';
            } else if (value.length > 0) {
                this.style.borderColor = 'var(--ubits-feedback-accent-error)';
                this.style.borderWidth = '2px';
            } else {
                this.style.borderColor = 'var(--ubits-border-1)';
                this.style.borderWidth = '1px';
            }
        });
    }
}, 500);
</script>

API Reference 

// FUNCIÓN PRINCIPAL
createInput(options)

// PARÁMETROS OBLIGATORIOS
{
    containerId: string  // ID del contenedor donde crear el input
}

// PARÁMETROS OPCIONALES
{
    label: string,                    // Texto del label
    placeholder: string,              // Placeholder del input
    type: 'text' | 'email' | 'password' | 'number' | 'tel' | 'url' | 'select' | 'textarea' | 'search' | 'autocomplete' | 'calendar',
    size: 'sm' | 'md' | 'lg',        // Tamaño del input (default: 'md')
    value: string,                    // Valor inicial
    leftIcon: string,                 // Icono izquierdo (FontAwesome)
    rightIcon: string,                // Icono derecho (FontAwesome)
    helperText: string,               // Texto de ayuda
    showHelper: boolean,              // Mostrar helper text
    maxLength: number,                // Límite de caracteres
    showCounter: boolean,             // Mostrar contador de caracteres
    mandatory: boolean,               // Mostrar texto mandatory
    mandatoryType: 'obligatorio' | 'opcional',  // Tipo de mandatory
    selectOptions: Array,             // Opciones para SELECT
    autocompleteOptions: Array,       // Opciones para AUTOCOMPLETE
    onChange: function,               // Callback al cambiar valor
    onFocus: function,                // Callback al enfocar
    onBlur: function                  // Callback al desenfocar
}

// MÉTODOS DISPONIBLES
const input = createInput({...});

input.getValue()          // Obtener valor actual
input.setValue(value)     // Establecer valor
input.focus()             // Enfocar input
input.blur()              // Desenfocar input
input.disable()           // Deshabilitar input
input.enable()            // Habilitar input
input.setState(state)     // Cambiar estado (default, hover, focus, invalid, disabled)

// REGLAS CRÍTICAS
// 1. SIEMPRE importar archivos requeridos
// 2. VALIDACIÓN MANUAL obligatoria para email, tel, url
// 3. POSICIONAMIENTO: contenedor debe tener position: relative para dropdowns
// 4. USAR tokens UBITS para colores y tipografía
// 5. ICONOS: usar FontAwesome con clase 'far' (outline)
// 6. NO crear componentes custom cuando existen UBITS
