Card content

Componente para mostrar contenidos de aprendizaje con información completa, progreso visual y diseño responsive.

Preview del componente

Personaliza el card en tiempo real usando los selectores. Cambia el estado, progreso, tipo de contenido y más para ver todas las variantes disponibles.

El componente Card Content está diseñado para mostrar contenidos de aprendizaje de manera profesional y consistente.

Responsive
Adaptable a todos los dispositivos con grid automático y breakpoints optimizados

Cómo implementar

🔧 REQUISITOS OBLIGATORIOS:

CSS Requerido:
• components/card-content.css - Estilos del componente
• fontawesome-icons.css - Iconos FontAwesome
• ubits-colors.css - Tokens de color UBITS
• ubits-typography.css - Tipografía UBITS

JavaScript Requerido:
• components/card-content.js - Función loadCardContent()

Renderizado: loadCardContent() function con containerId y datos 
<!-- 1. HTML: Crear contenedor -->
<div id="mi-contenedor-cards"></div>

<!-- 2. CSS: Importar dependencias -->
<link rel="stylesheet" href="ubits-colors.css">
<link rel="stylesheet" href="ubits-typography.css">
<link rel="stylesheet" href="fontawesome-icons.css">
<link rel="stylesheet" href="components/card-content.css">

<!-- 3. JavaScript: Cargar componente -->
<script src="components/card-content.js"></script>

<!-- 4. Implementación: Usar el componente -->
<script>
document.addEventListener('DOMContentLoaded', function() {
    const datosCards = [
        {
            type: 'Curso',
            title: 'Mi contenido',
            provider: 'UBITS',
            providerLogo: 'images/Favicons/UBITS.jpg',
            duration: '60 min',
            level: 'Intermedio',
            progress: 50,
            status: 'progress',
            image: 'images/cards-learn/mi-imagen.jpg',
            competency: 'Product design',
            language: 'Español'
        }
    ];
    
    loadCardContent('mi-contenedor-cards', datosCards);
});
</script>

API y funciones

loadCardContent(containerId, cardsData)

Carga múltiples cards de contenido en un contenedor específico con grid responsive automático.

Parámetros: containerId (string), cardsData (array)
Ejemplo: loadCardContent('mi-contenedor', datosArray)

renderCardContent(cardData)

Renderiza una sola card de contenido y devuelve el HTML como string para insertar manualmente.

Parámetros: cardData (object) - Datos de la card
Ejemplo: renderCardContent(miCardData)

validateCardData(cardData)

Valida que los datos de la card tengan tipos y competencias válidas según las opciones oficiales UBITS.

Parámetros: cardData (object) - Datos a validar
Ejemplo: validateCardData(miCardData)

getRecommendedDuration(type)

Obtiene la duración recomendada según el tipo de contenido (Cápsula = 15min, Ruta = 240min, etc.).

Parámetros: type (string) - Tipo de contenido
Ejemplo: getRecommendedDuration('Cápsula') // '15 min'

window.CARD_CONTENT_OPTIONS

Objeto global que contiene todas las opciones disponibles (tipos, competencias, niveles, proveedores, etc.).

Uso: Variable global de solo lectura
Ejemplo: console.log(window.CARD_CONTENT_OPTIONS)

Personalización

Agregar nuevos tipos de contenido

Para agregar nuevos tipos de contenido, modifica el array CONTENT_TYPES en el archivo JavaScript: 

// En card-content.js, línea 11-23
const CONTENT_TYPES = [
    'Curso',
    'Cápsula', 
    'Charla',
    'Artículo',
    'Podcast',
    'Libro',
    'Ideas de libro',
    'Caso de estudio',
    'Documento técnico',
    'Ejercicios de práctica',
    'Ruta de aprendizaje',
    'Mi Nuevo Tipo'  // ← Agregar aquí
];

Personalizar estilos CSS

Puedes personalizar la apariencia del componente sobrescribiendo las clases CSS: 

/* Personalizar el hover del card */
.course-card:hover {
  transform: translateY(-4px); /* Más elevación */
  box-shadow: 0 8px 24px rgba(0, 0, 0, 0.15); /* Sombra más fuerte */
}

/* Personalizar el tamaño de la imagen */
.course-thumbnail {
  aspect-ratio: 4 / 3; /* Cambiar de 16:9 a 4:3 */
}

/* Personalizar colores de progreso */
.course-progress-overlay .progress-fill {
  background: var(--ubits-feedback-accent-warning-static); /* Naranja */
}

/* Personalizar tipografía del título */
.course-title {
  font-size: 18px; /* Título más grande */
  line-height: 1.4;
}
