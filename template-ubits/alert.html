Cómo implementar

🔧 REQUISITOS OBLIGATORIOS:

CSS Requerido:
• components/alert.css - Estilos del componente
• fontawesome-icons.css - Iconos FontAwesome

JavaScript (opcional):
• components/alert.js - Solo para función showAlert()

Renderizado: HTML directo O función showAlert()
Implementación básica 
<!-- OPCIÓN 1: HTML Directo (Recomendado para alerts estáticos) -->
<div class="ubits-alert ubits-alert--success ubits-alert--no-close">
  <div class="ubits-alert__icon">
    <i class="far fa-check-circle"></i>
  </div>
  <div class="ubits-alert__content">
    <div class="ubits-alert__text">Tu mensaje aquí</div>
  </div>
</div>

<!-- OPCIÓN 2: JavaScript Component (Recomendado para alerts dinámicos) -->
<div id="alert-container"></div>
<script>
showAlert('success', 'Tu mensaje aquí', {
  containerId: 'alert-container',
  noClose: true
});
</script>

<!-- CSS REQUERIDO -->
<link rel="stylesheet" href="components/alert.css">

<!-- JavaScript REQUERIDO (solo para Opción 2) -->
<script src="components/alert.js"></script>

API y funciones

showAlert(type, message, options)

Función helper para crear alerts fácilmente. Requiere que el CSS y JS del componente estén importados.

Parámetros:
• type: 'success', 'info', 'warning', 'error'
• message: Texto del alert (puede incluir HTML básico)
• options: { containerId: 'id', noClose: true/false }

Ejemplo básico:
showAlert('warning', 'Tu mensaje aquí', { containerId: 'mi-container', noClose: true });

Ejemplo con HTML:
showAlert('info', 'Mensaje con <strong>texto destacado</strong>', { containerId: 'container' });
