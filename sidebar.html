Sidebar

Componente de navegación lateral que proporciona acceso rápido a las diferentes secciones de la aplicación. Diseñado para desktop con funcionalidad responsive.

Preview del componente

Aquí puedes ver el componente Sidebar en acción. Haz clic en los botones para ver los estados activos y hover.

Características generales

Desktop
Fijo lateral
Móvil
Oculto (se reemplaza por el tab-bar)
Temas
Claro/Oscuro (mismos colores en ambos)

Prompts útiles

Usa estos prompts con Cursor AI para configurar y personalizar el componente Sidebar de manera eficiente.

Activar sección del sidebar 

En la página [nombre-pagina.html] en el componente SIDEBAR UBITS (navegación lateral izquierda) haz que se vea activa la sección de [opciones: aprendizaje, diagnóstico, desempeño, encuestas, reclutamiento, tareas, ubits-ai] activa. Usa la función loadSidebar().

Cambiar sección activa 

En la página [nombre-pagina.html], cambia la sección activa del sidebar de [sección actual] a [nueva sección que quieres activar].

Ocultar sidebar en página específica 

En la página [nombre-pagina.html], oculta el sidebar pero mantén el TabBar para responsive. Cambia el margin-left del main-content de 131px a 12px para que el contenido ocupe todo el ancho disponible.

<!-- CSS -->
<link rel="stylesheet" href="styles.css">

<!-- JavaScript -->
loadSidebar('desempeño'); // Cargar sidebar con sección activa
updateActiveSidebarButton('desempeño'); // Cambiar botón activo

API y funciones

loadSidebar(activeButton)

Carga el componente sidebar en el contenedor 'sidebar-container'. Busca automáticamente el contenedor y activa la sección especificada.

Parámetros: activeButton (string, opcional) - Sección a activar
Ejemplo: loadSidebar('aprendizaje')
Sin parámetro: loadSidebar() - Ninguna sección activa

updateActiveSidebarButton(activeButton)

Actualiza el botón activo del sidebar sin recargar el componente.

Parámetros: activeButton (string) - ID del nuevo botón activo
Ejemplo: updateActiveSidebarButton('desempeño')

Personalización

Agregar nuevos botones

Para agregar nuevos botones de navegación, edita el HTML del sidebar en components/sidebar.js: 

// En sidebar.js, dentro del sidebarHTML (línea 45-66)
<button class="nav-button" data-section="nueva-seccion" data-tooltip="Nueva Sección">
  <i class="far fa-nuevo-icono"></i>
</button>

Modificar estilos

Los estilos se pueden personalizar editando las variables CSS en styles.css: 

/* Personalizar colores del sidebar */
.sidebar {
  background-color: var(--ubits-sidebar-bg);
  width: 96px;
  height: 578px;
}

/* Personalizar botones de navegación */
.nav-button {
  background: rgba(255, 255, 255, 0);
  border: none;
  padding: 8px;
  border-radius: 1000px;
  cursor: pointer;
}
