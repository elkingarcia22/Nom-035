Toast

Mensajes breves que aparecen arriba, al centro, para informar algo rápido al usuario. Se muestran por unos segundos y luego se van solos.

Preview del componente

Prueba las variantes principales del toast. Los mensajes aparecen arriba, al centro, y se cierran solos.

Características generales

Toasts ligeros y consistentes con UBITS: posición superior centrada, cierre automático y accesibilidad integrada.

Posición y apilado
Siempre aparecen arriba y al centro, uno debajo de otro, para que no te tapes nada importante.
Duración
Se cierran solos: success e info ~3.5 segundos, warning ~5 segundos y error ~6.5 segundos.
Pausa en hover y foco
Si pasas el cursor o navegas con teclado, se quedan un momento más para que puedas leerlos con calma.

Cómo implementar

🔧 REQUISITOS OBLIGATORIOS:

CSS Requerido:
• components/toast.css - Estilos del componente
• fontawesome-icons.css - Iconos FontAwesome
• ubits-colors.css - Tokens de color UBITS

JavaScript Requerido:
• components/toast.js - Función showToast()

Container Requerido:
• <div id="ubits-toast-container"></div> - Container obligatorio

Renderizado: showToast() function 
<!-- 1. IMPORTAR ARCHIVOS NECESARIOS -->
<link rel="stylesheet" href="ubits-colors.css">
<link rel="stylesheet" href="fontawesome-icons.css">
<link rel="stylesheet" href="components/toast.css">
<script src="components/toast.js"></script>

<!-- 2. CREAR CONTENEDOR -->
<div id="ubits-toast-container"></div>

<!-- 3. USAR LA FUNCIÓN -->
<script>
// Toast básico
showToast('success', '¡Operación exitosa!');

// Toast con opciones
showToast('info', 'Ya estás en la documentación 😆', {
    containerId: 'ubits-toast-container',
    duration: 3500,
    noClose: false
});

// Toast con botón de acción
showToast('warning', 'Datos no guardados', {
    action: { 
        label: 'Guardar', 
        onClick: () => save() 
    }
});
</script>

API y funciones

showToast(type, message, options)

Función helper para crear toasts fácilmente. Requiere que el CSS y JS del componente estén importados.

Parámetros:
• type: 'success' (3.5s), 'info' (3.5s), 'warning' (5s), 'error' (6.5s)
• message: Texto del toast (string)
• options: { containerId, duration, noClose, pauseOnHover, action }

Opciones disponibles:
• containerId: string (ID del contenedor, default: 'ubits-toast-container')
• duration: number (milisegundos, 0 = persistente)
• noClose: boolean (ocultar botón cerrar)
• pauseOnHover: boolean (pausar en hover, default: true)
• action: {label: string, onClick: function} (botón de acción)

Ejemplos:
showToast('success', '¡Guardado!');
showToast('error', 'Error al guardar', { noClose: true });
showToast('warning', 'Datos no guardados', { action: { label: 'Guardar', onClick: save } });

Características y mejores prácticas

✅ Características

  • Auto-cierre después del tiempo especificado
  • Pausa en hover y foco para mejor UX
  • Apilado inteligente (máximo 3 visibles)

💡 Mejores prácticas

  • Success para confirmaciones

  • Warning para advertencias
  • Info para información general
  • No abuses de los toasts

Personalización

Los estilos se pueden personalizar editando las variables CSS en components/toast.css. Los toasts usan tokens UBITS para mantener consistencia con el sistema de diseño.

Variables CSS personalizables 
/* Personalizar colores de toasts */
.ubits-toast--success {
  background-color: var(--ubits-feedback-bg-success-subtle);
  border-color: var(--ubits-feedback-border-success-subtle);
  color: var(--ubits-feedback-fg-success-subtle);
}

.ubits-toast--info {
  background-color: var(--ubits-feedback-bg-info-subtle);
  border-color: var(--ubits-feedback-border-info-subtle);
  color: var(--ubits-feedback-fg-info-subtle);
}

.ubits-toast--warning {

  border-color: var(--ubits-feedback-border-warning-subtle);
  color: var(--ubits-feedback-fg-warning-subtle);

Solución de problemas

❌ Problemas comunes

  • Toast no aparece: Verifica que el contenedor tenga ID correcto
  • Sin estilos: Importa components/toast.css
  • Sin iconos: Importa fontawesome-icons.css
  • Colores incorrectos: Importa ubits-colors.css
  • No se cierra: Verifica que el JS esté cargado

✅ Checklist de verificación

  • ✓ Contenedor con ID ubits-toast-container
  • ✓ CSS: components/toast.css importado
  • ✓ JS: components/toast.js importado
  • ✓ Tokens: ubits-colors.css importado
  • ✓ Iconos: fontawesome-icons.css importado
  • ✓ Función: showToast() disponible
