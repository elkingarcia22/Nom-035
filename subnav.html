SubNav

Componente de navegación superior que muestra las sub-navegaciones de los módulos principales del sidebar.

Preview del componente

Aquí puedes ver el componente SubNav en acción. Haz clic en los botones para ver las diferentes variantes disponibles.

Mostrar

Estructura del componente 

<nav class="sub-nav" id="sub-nav">
  <div class="sub-nav-container">
    <div class="sub-nav-content">
      <div class="sub-nav-tabs">

          <i class="far fa-home"></i>
          <span>Inicio</span>
        </button>
        <button class="sub-nav-tab" data-section="componentes">
          <i class="far fa-cube"></i>
          <span>Componentes</span>
        </button>
        <!-- Más pestañas -->
      </div>
    </div>
  </div>
</nav>

Cómo implementar 

<!-- HTML -->
<div id="sub-nav-container"></div>

<!-- CSS -->
<link rel="stylesheet" href="components/sub-nav.css">

<!-- JavaScript -->
loadSubNav('sub-nav-container', 'template'); // Cargar SubNav con variante template
updateActiveSubNavTab('componentes'); // Cambiar pestaña activa

API y funciones

loadSubNav(containerId, variant)

Carga el componente SubNav en el contenedor especificado con la variante seleccionada.

Parámetros: containerId (string) - ID del contenedor, variant (string) - Variante del SubNav
Ejemplo: loadSubNav('sub-nav-container', 'template')

updateActiveSubNavTab(tabId)

Actualiza la pestaña activa del SubNav sin recargar el componente.

Parámetros: tabId (string) - ID de la nueva pestaña activa
Ejemplo: updateActiveSubNavTab('componentes')

Personalización

Agregar nuevas funcionalidades

Para agregar nuevas pestañas, edita el HTML del SubNav en components/sub-nav.js: 

// En sub-nav.js, dentro del subNavHTML (línea 45-66)
<button class="sub-nav-tab" data-section="nueva-seccion">
  <i class="far fa-nuevo-icono"></i>
  <span>Nueva Sección</span>
</button>

Modificar estilos

Los estilos se pueden personalizar editando las variables CSS en components/sub-nav.css: 

/* Personalizar colores del SubNav */
.sub-nav {
  background-color: var(--ubits-bg-1);
  border-bottom: 1px solid var(--ubits-border-1);
}

/* Personalizar pestañas activas */
.sub-nav-tab.active {
  color: var(--ubits-accent-brand);
  border-bottom: 2px solid var(--ubits-accent-brand);
}
